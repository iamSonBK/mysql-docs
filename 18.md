# 18 - MYSQL BACKEND(18): Giám sát (Monitoring) MySQL ONLINE qua hệ thống API chuyên dụng

## Công cụ giám sát hiệu năng
[cite_start]Việc tận dụng các tính năng hiện đại như Performance Schema để giám sát thời gian thực sẽ giúp xây dựng các hệ thống backend mạnh mẽ[cite: 89].

[cite_start]Lệnh `EXPLAIN` cung cấp thông tin về cách bảng được kết nối, chỉ mục nào được chọn và ước tính số lượng hàng phải kiểm tra[cite: 52].
* [cite_start]Các cảnh báo đỏ trong cột Extra: `Using filesort` (sắp xếp chậm), `Using temporary` (bảng tạm tốn kém)[cite: 57].
* [cite_start]Cột `rows`: Số hàng ước tính phải quét, giá trị càng nhỏ càng tốt[cite: 56].
